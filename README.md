# Woodthrush
Woodthrush is a simple forum software built on Blazor. It is minimal and highly-customizable.
## Specifications
- Runs on .NET 8
- Uses SQLite as database
## Features
- Categorized discussions
- Personal messaging
- For admins: manage threads, users,...
- Customizable UI with themes
